Core Gameplay

Offering Tube （御供筒）

The core mechanic is an offering tube . It works like a stack system （栈结构）.

  • Items must be stored in the container.
  • Once stored, items cannot be changed.
  • If an item is not stored in the bag, it disappears （不放进背包就会消失）.
  • The bag uses Last In, First Out .
  • The altar （祭坛） reads items from bottom to top.
  • The player must collect items in reverse order.
Game Loop

Explore → Collect → Choose → Store → Complete ritual → Unlock next truth

If an item is not stored in the bag, it disappears （不放进背包就会消失）. That upgrades the mechanic from sorting to sorting plus trade-off.

Level Design

Level 1

Tutorial .

Follow one clear route, collect the first offerings, and reach the altar.

A split path forces a choice: if an item is not stored in the bag, it disappears.

Technology & Design Lens
09 / 09

Technology

Prototype Need
  • 2D traversal through shrine spaces and room-to-room progression.
  • A stack-based bag that locks order once items are stored.
  • State tracking for missing items, completed rituals, and unlocked truths.
Why It Fits

The concept does not need huge content or complex combat. The technology supports sequence, choice, and consequence in a student-sized prototype.

Elemental Tetrad

Four Elements
  • Mechanics: ordering, storing, losing items, and completing rituals.
  • Story: a shrine apprentice slowly learns the ritual is morally wrong.
  • Aesthetics: formal beauty, sacred spaces, and quiet unease.
  • Technology: a lightweight 2D system that expresses sequence, choice, and consequence.
